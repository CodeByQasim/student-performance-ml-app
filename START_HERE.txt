🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

1. GET /
   └─ Returns main HTML page

2. POST /api/predict
   ├─ Request: JSON with study_hours, phone_hours, sleep_hours, attendance
   ├─ Response: Prediction, confidence, insights
   ├─ Status: 200 (success), 400 (validation error), 503 (model unavailable)
   └─ Example: curl -X POST http://localhost:5000/api/predict \
                  -H "Content-Type: application/json" \
                  -d '{"study_hours": 6, "phone_hours": 3, "sleep_hours": 8, "attendance": 85}'

3. GET /api/tips
   ├─ Response: Array of success tips
   ├─ Status: 200
   └─ Example: curl http://localhost:5000/api/tips

4. GET /api/health
   ├─ Response: Health status and model availability
   ├─ Status: 200
   └─ Example: curl http://localhost:5000/api/health
